# R2Z1-03_Code
Repository containing all the code included in our research project "Development, Modification and Comparison of Two Neural Networks for Classifying Selected Fabaceae Family Plant Leaves and Flowers." Each commit is tracked and recorded for changes in the code, which allows for easy access to previous code for the CNNs from past iterations during the project's implementation.
